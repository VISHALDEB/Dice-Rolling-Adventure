# 🎲 Dice Rolling Adventure

A simple **Java-based console game** where players roll dice to reach a target score.
This project helped me understand core programming concepts like OOP, loops, and user interaction.

---

## 🚀 Features

* 🎯 User-defined target score
* 🎲 Random dice rolling (2 dice)
* ⭐ Bonus points for rolling doubles
* 🔁 Continuous gameplay loop
* 🧠 Input handling using Scanner

---

## 🛠️ Tech Stack

* Java
* OOP Concepts (Classes, Objects, Methods)
* Java Collections & Random

---

## 📂 Project Structure

```
Dice.java   → Handles dice logic (roll, value)
Game.java   → Manages game rules and score
Main.java   → Handles user input and game flow
```

## 🧠 What I Learned

While building this project, I learned:

* How to structure a program using **multiple classes**
* The importance of **separation of concerns**

  * `Dice` → handles dice behavior
  * `Game` → controls logic
  * `Main` → handles user interaction
* How to use **Random class** for generating values
* How to manage **loops and conditions** for game flow
* How to take **user input using Scanner**
* Basic **debugging and code organization**

---

## 💡 My Understanding

This project helped me understand how real-world applications are structured:

* Breaking a problem into smaller parts makes coding easier
* Each class should have a **specific responsibility**
* Clean and readable code improves maintainability
* Even a simple game requires proper logic design
